# Project 1: Excel Data Cleaning

## Project Overview

This project focuses on cleaning and preparing a raw sales dataset using Microsoft Excel. The objective was to improve the quality of the data before analysis by identifying and correcting errors, handling missing values, removing duplicates, and standardizing the dataset.

## Objectives

- Clean and prepare raw data for analysis.
- Improve data accuracy and consistency.
- Create a structured dataset ready for reporting and visualization.

## Tools Used

- Microsoft Excel

## Dataset

The dataset contains sales transaction records, including:

- Order ID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Shipping Address
- Payment Method
- Order Status
- Tracking Number
- Items in Cart
- Coupon Code
- Referral Source
- Total Price

## Data Cleaning Tasks Performed

- Removed duplicate records.
- Checked for missing values.
- Corrected inconsistent data entries.
- Standardized text formatting.
- Formatted dates correctly.
- Verified numeric columns.
- Ensured column headers were accurate.
- Prepared the dataset for analysis.
